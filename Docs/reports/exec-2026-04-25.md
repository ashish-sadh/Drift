## What Shipped This Period

- **5-tier test architecture** — Tests reorganized by cost tier: macOS `swift test` (Tier 0, ~0.1s) for logic, iOS simulator (Tier 1, ~30s) for UI/HealthKit, LLM eval (Tier 2/3) for AI pipeline. Each file belongs to exactly one tier.
