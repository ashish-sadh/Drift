# Drift Daily Briefing — 2026-04-15

## Headline
Smart unit defaults now cover 2,046+ foods — users logging dosa, gulab jamun, and milk no longer see grams by default.

## Product Snapshot
Drift is an AI-first health tracker — users type what they ate or did, AI handles logging, tracking, and insights. On-device, no cloud. Currently in closed beta on TestFlight.

## What Shipped This Period
- **Smart units now intelligently defaults across 30+ new food categories** — batters get cups, sweets get pieces, powders get scoops; no more grams-by-default for foods that aren't measured in grams
- **Food database grew to 2,046 items** — added Filipino, Turkish, and Ethiopian cuisines alongside Indian regional expansion
- **Custom macro goal targets now calculate correctly** — when users set all three macros manually, the calorie target reflects the actual macro sum, not an estimate; dashboard and goal views are now consistent
- **Goal setup shows honest weight projections** — when custom macros are set, timeline now shows actual intake-vs-TDEE math ("your macros put you in a 160 kcal surplus") instead of a geometric estimate
- **AI lab report parsing** — users can photograph or paste lab results; AI extracts biomarkers with chunked extraction

## What's Working Well
- Smart units saturation is accelerating — 30+ rules added in a single batch with 25 automated regression tests catching regressions
- Test suite health is strong: 1,539 tests, all green, covering both AI intent and pure logic
- Custom macro feature shipped clean — four bugs caught and fixed within the same session before user-visible release

## Risks & Blockers
- **Food database at 2,046 vs MFP's 20M** — coverage gaps will surface for any user outside Indian/Asian cuisine comfort zone; each sprint adds ~21 foods which won't close the gap at this rate
- **Smart units still incomplete on recipe builder interface** — portionText sync is done but the recipe builder entry path needs its own pass
- **Metal GPU broken on A19 Pro** — CPU-only mode for newest hardware; AI responses are slower on the latest iPhones

## Focus for Next Period
1. **Smart units: recipe builder + manual entry** — the food search path is largely covered; the recipe builder and manual add-food sheet still show grams by default for foods like dosa batter and gulab jamun
2. **AI chat LLM pipeline validation** — Stage 3b validation is shipped but real-world query failure rate from `Docs/failing-queries.md` needs a systematic pass
3. **Food DB coverage** — continue +21/sprint cadence; prioritize foods that surface from AI parse failures
